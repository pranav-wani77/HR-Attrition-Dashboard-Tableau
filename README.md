# HR-Attrition-Dashboard-Tableau
This project focuses on analyzing employee attrition and workforce trends using Tableau. The dashboard helps HR teams understand why employees leave, identify high-risk groups, and make data-driven retention decisions.

Project Overview
This project focuses on analyzing employee attrition and workforce trends using Tableau.
The dashboard helps HR teams understand why employees leave, identify high-risk groups, and make data-driven retention decisions.

Objectives
Analyze overall employee attrition rate
Identify attrition trends by department, gender, age group, and education
Evaluate job satisfaction levels across roles
Provide interactive insights for HR decision-making

Key KPIs
Total Employees: 1,470
Attrition Count: 237
Attrition Rate: 16.12%
Active Employees: 1,233
Average Age: 37
